# 사이드 패널 간소화 + 기간 컬럼 — 설계 문서

## 개요

사이드 패널을 업무 집중 모드로 재구성한다. 속성 사이드바/서브태스크/첨부파일 섹션을 제거하고, 체크리스트와 활동 로그 중심의 단일 컬럼 레이아웃으로 변경한다. 리스트 뷰에서는 마감일 컬럼을 시작일~마감일 통합 "기간" 컬럼으로 교체한다.

## 결정사항

| 항목 | 결정 |
|---|---|
| 패널 속성 사이드바 | 제거 (리스트에서 인라인 편집 가능하므로 불필요) |
| 패널 레이아웃 | 단일 컬럼: 제목 → 설명 → 체크리스트 → 활동 로그 |
| 서브태스크 | 패널에서 제거 (체크리스트로 통합) |
| 첨부파일 섹션 | 패널에서 제거 (전체 페이지 `/tasks/[id]` 에서만 관리) |
| 활동 입력 | 드래그앤드롭 + Ctrl+V 파일 첨부 지원 |
| 리스트 뷰 마감일 | "기간" 컬럼으로 변경 (시작일~마감일 통합 표시 + 인라인 편집) |
| 기존 상세 페이지 | 그대로 유지 (속성, 첨부파일, 서브태스크 등 전체 기능 접근 가능) |

## 1. 패널 레이아웃 변경

### 현재 (2단 컬럼)
```
┌─ 본문 (2/3) ──────────┬─ 사이드바 (1/3) ─┐
│ 제목                   │ 상태             │
│ 설명                   │ 우선순위         │
│ 체크리스트             │ 카테고리         │
│ 서브태스크             │ 시작일/마감일    │
│ 활동 로그              │ 담당자           │
│ 댓글 입력              │ 생성자/생성일    │
│                        │ 첨부파일         │
└────────────────────────┴──────────────────┘
```

### 변경 후 (단일 컬럼)
```
┌─ 패널 전체 ────────────────────────────────┐
│ [X 닫기]                        [저장] [🗑] │
│                                             │
│ 제목 (편집 가능)                            │
│ 설명 (편집 가능)                            │
│                                             │
│ ── 체크리스트 ────────────────────────────  │
│ ☑ 항목 1                                   │
│ ☐ 항목 2                                   │
│ + 항목 추가                                 │
│                                             │
│ ── 활동 ──────────────────────────────────  │
│ [타임라인]                                  │
│ ┌─────────────────────────────────────────┐ │
│ │ 댓글 입력...          [📎 드래그/붙여넣기]│ │
│ └─────────────────────────────────────────┘ │
└─────────────────────────────────────────────┘
```

### 변경 파일
- `src/components/dashboard/tasks/detail/TaskDetailClient.tsx` — `mode === "panel"` 일 때 단일 컬럼 렌더링. 속성/서브태스크/첨부파일 섹션 미표시.

## 2. 활동 입력 파일 첨부

### 기능
- **드래그앤드롭**: 활동 입력 영역에 파일을 드래그하면 첨부
- **Ctrl+V (붙여넣기)**: 클립보드의 이미지/파일을 붙여넣기로 첨부
- 첨부된 파일은 Supabase Storage에 업로드 후 활동(comment)에 메타데이터로 포함
- 첨부 미리보기: 이미지는 썸네일, 그 외 파일은 파일명+크기 표시
- 여러 파일 동시 첨부 가능

### 변경 파일
- `src/components/dashboard/tasks/detail/TaskCommentInput.tsx` — 드래그앤드롭/붙여넣기 이벤트 핸들링, 파일 업로드 로직, 미리보기 UI 추가

### 업로드 흐름
```
파일 드롭/붙여넣기
  → 미리보기에 파일 추가 (로컬 상태)
  → 댓글 전송 시:
    1. 파일들을 Supabase Storage에 업로드 (task-attachments 버킷)
    2. 업로드된 파일 URL을 comment metadata에 포함
    3. task_activities 테이블에 activity 생성 (type: "comment", metadata: { attachments: [...] })
  → 활동 타임라인에서 첨부 포함 댓글로 표시
```

### 활동 타임라인 표시
- `TaskActivityTimeline.tsx`에서 comment의 metadata.attachments가 있으면 댓글 아래에 첨부 파일 목록/썸네일 렌더링

## 3. 리스트 뷰 기간 컬럼

### 현재
- "마감일" 컬럼: `due_date`만 표시, 클릭 시 date input으로 편집

### 변경 후
- "기간" 컬럼: `start_date ~ due_date` 형태로 표시
  - 시작일만 있으면: `04.10 ~`
  - 마감일만 있으면: `~ 04.17`
  - 둘 다 있으면: `04.10 ~ 04.17`
  - 둘 다 없으면: `-`
- 클릭 시: 시작일과 마감일 두 개의 date input이 나오는 드롭다운
- 마감일 지연 표시(빨간색 등)는 기존 로직 유지

### 변경 파일
- `src/components/dashboard/tasks/views/ListRow.tsx` — 마감일 컬럼을 기간 컬럼으로 교체, 편집 드롭다운에 시작일/마감일 두 input
- `src/components/dashboard/tasks/views/ListView.tsx` — 테이블 헤더 "마감일" → "기간"으로 변경

## 4. 범위 외 (변경 없음)

- 기존 상세 페이지 (`/tasks/[id]`): 모든 기능 그대로 유지 (속성, 서브태스크, 첨부파일 등)
- page 모드의 `TaskDetailClient`: 기존 2단 레이아웃 유지
- DB 스키마: 변경 없음 (기존 테이블/컬럼 그대로 사용)
